How to Play

Make a word using the given two letters to get points. The longer the word you make, the more points you get. Try to get as many points per guess as possible!

Scoring Chart

First 5 letters (1 - 5): 1 point each
Next 5 letters (6 - 10): 2 points each
Next 5 letters (11 - 15): 4 points each
All letters after (15+): 8 points each
